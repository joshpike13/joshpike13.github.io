DREAM Data Analysis Tool

CS 426 Senior Project

Spring 2018

UNR CSE Department

The DREAM challenge is a non-profit effort that provides extensive sets of data to the public for universities and researchers to analyze and discover new relations in the data. The challenge organizers have proposed three problems for a competition that will allow participants to view and analyze large amounts of data from The Clinical Proteomic Tumor Analysis Consortium and discover any possible relations between DNA and various proteins. These analyses may shed light on the behavior of cancer and be beneficial to the field of cancer research.

Team 30 is developing a software solution to the competition by creating a machine learning algorithm to solve the problems posed by the challenge. The DREAM challenge has already ended too soon for Team 30 to compete, but the sponsors have given the team permission to use their data outside of the competition.

Team 30 has begun development on a user interface for uploading and visualizing data. The interface was created using CSS and Flask, a framework for web apps using Python. On the data analytic back end, several algorithms have been chosen that have shown promise in imputing missing proteomic values, which is the first problem of the DREAM challenge. Work on the interface between the web page and back end is ongoing, but enough of each side has been developed to create a prototype demonstrating some of the intended functionality of the final project.

Team 30 has decided to create a browser-based solution rather than a desktop application. The functionality will remain the same and the data analysis will run on the client host.
